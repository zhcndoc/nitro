## 使用 git 部署

<steps level="4">

#### 将您的代码推送到 git 仓库（当前仅支持 GitHub）。

#### Zeabur 将检测您正在使用 Nitro，并为您的部署启用正确的设置。

#### 您的应用程序已成功部署！

</steps>
