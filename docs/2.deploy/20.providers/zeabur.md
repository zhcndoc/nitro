# Zeabur

> 将 Nitro 应用部署到 [Zeabur](https://zeabur.com)。

**预设：** `zeabur`

:read-more{title="Zeabur" to="https://zeabur.com"}

::note
使用此提供商进行集成可以通过[零配置](/deploy/#zero-config-providers)实现。
::

## 使用 Git 部署

1. 将你的代码推送到 Git 仓库（目前仅支持 GitHub）。
2. 将你的项目[导入](https://zeabur.com/docs/get-started)到 Zeabur。
3. Zeabur 将检测到你正在使用 Nitro，并为你的部署启用正确的设置。
4. 你的应用已部署完成！
