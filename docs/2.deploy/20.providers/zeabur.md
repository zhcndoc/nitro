# Zeabur

> 将 Nitro 应用部署到 [Zeabur](https://zeabur.com)。

**预设:** `zeabur`

:read-more{title="Zeabur" to="https://zeabur.com"}

::note
与此提供商的集成可以通过 [零配置](/deploy/#zero-config-providers) 实现。
::

## 使用 git 部署

1. 将您的代码推送到 git 仓库（当前仅支持 GitHub）。
2. [将您的项目导入](https://zeabur.com/docs/get-started) 到 Zeabur。
3. Zeabur 将检测您正在使用 Nitro，并为您的部署启用正确的设置。
4. 您的应用程序已成功部署！
