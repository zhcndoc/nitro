快速开始

从一个全新的 Nitro 项目开始，或者将它集成到现有的 Vite 项目中。
Nitro v3 Alpha 文档仍在开发中 — 请预计会有更新、尚未打磨的部分以及偶尔的不准确之处。

在线体验 Nitro

在浏览器中使用我们的 playground 体验 Nitro。

Nitro+Vite 入门

在浏览器中使用一个极简的 Vite 项目玩转 Nitro。

创建 Nitro 项目

创建 Nitro 应用最快捷的方式是使用 create-nitro-app。

!注意 请确保已安装最新的 Node.js、Bun 或 Deno 的长期支持版本（LTS）。 Node.js、Bun、Deno

npx create-nitro-app
yarn dlx create-nitro-app
pnpm dlx create-nitro-app
bunx create-nitro-app
deno run -A npm:create-nitro-app

按照命令行界面的指引操作，即可开始启动开发服务器。

添加到 Vite 项目

若要将 Nitro 添加到现有 Vite 项目，请按照以下步骤：

安装 nitro 包

npm i nitro
yarn add nitro
pnpm i nitro
bun i nitro
deno i npm:nitro

添加 Nitro 插件

vite.config.mjs
import { defineConfig } from "vite";
import { nitro } from "nitro/vite";

export default defineConfig({
  plugins: [
    nitro()
  ],
  nitro: {
    serverDir: './'
  }
});

就是这么简单，现在你可以向你的 Vite 项目添加服务器和 API 路由了！

介绍

Nitro 是一个全栈框架，兼容任何运行时。它扩展了你的 Vite 应用，使其具备生产环境的服务器能力。
